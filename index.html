¿Cómo se calcula?

  • De 0kg a 10kg, se calcula 100cc por cada kilo.
  • Se suman 50 cc por cada kilo de peso pro arriba de 10kg, hasta 20kg
  • De 29kg para arriba se suman 20cc por cada kilo adiconal
  • Si el paciente pesa mas de 30kg se utiliza el metodo de calculo por Superficie Corporal
